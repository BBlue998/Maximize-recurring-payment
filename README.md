# 공유 오피스 유료 결제 전환율 예측: 머신러닝 기반 비즈니스 제안

**분석 기간:** 2025.12.23 ~ 2026.01.07  
**팀원:** 노유림, 세빌, 이재영, 정담연   
**사용 도구:** Python (Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn)

---

## 프로젝트 개요

공유 오피스 3일 체험 이용자의 행동 로그를 기반으로, 유료 결제 전환 가능성이 있는 고객을 사전에 예측하는 분류 모델을 구축한 프로젝트입니다. 클래스 불균형 환경에서 결제 고객을 놓치지 않는 것을 우선 목표로 Recall 중심으로 설계하였습니다.

---

## 데이터 개요

- 공유 오피스 서비스 약 2년 8개월치 방문 로그 데이터 (2021.05 ~ 2023.12)
- 비결제 5,995명 / 결제 3,664명 (클래스 불균형 구조)
- 체류 시간 합계·평균·최소·최대, 방문 일수, 지점 정보 등 파생 변수 생성

---

## 분석 과정

- **모델 비교** — Decision Tree, Random Forest, XGBoost 후보 중 Recall 기준으로 Random Forest 최종 선택
- **하이퍼파라미터 튜닝** — max_depth, min_samples_split 등 조정으로 과적합 억제 및 Recall 향상
- **Threshold 조정** — 기본값 0.5 → 0.4로 조정, Recall 0.63 → 0.92 / FN 175건 → 38건으로 개선
- **Confusion Matrix 기반 EDA** — TP/FP/FN/TN 유형별 행동 특성 분석, FN(고관여·신중형 결제 고객) 별도 식별

---

## 핵심 발견

- 체류 시간이 길수록 결제 가능성이 낮아지는 역설적 패턴 확인 (직관과 반대)
- 결제 고객은 짧은 체험 초기 구간에서 빠르게 결제 시그널을 보이는 경향
- FN 고객은 장시간 탐색 후 결제하는 고관여형으로, 별도 장기 설득 전략이 필요한 세그먼트

---

## 비즈니스 제안

- 체험 초기 단계에서 결제 가능성이 높은 고객에게 즉시 결제 유도 메시지·한정 프로모션 집중 노출
- 장시간 체류 고객(FN 유형)에게는 즉각 할인보다 정보 제공·비교 콘텐츠 중심의 장기 설득 전략 분리 적용
